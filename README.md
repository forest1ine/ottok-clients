# Прогнозирование оттока клиентов банка (Customer Churn Prediction)

## 📋 Описание проекта

Проект посвящен анализу оттока клиентов международного банка и разработке моделей машинного обучения для прогнозирования вероятности ухода клиента. 

**Актуальность:** Привлечение нового клиента обходится банку дороже, чем удержание существующего. Своевременное выявление клиентов, склонных к уходу, позволяет разрабатывать программы лояльности и снижать отток.

## 🎯 Цели проекта

1. Провести разведочный анализ данных и выявить факторы, влияющие на отток клиентов
2. Составить портрет клиента, склонного к уходу из банка
3. Разработать и сравнить две модели машинного обучения для прогнозирования оттока
4. Подготовить рекомендации для отдела маркетинга

## 📊 Описание данных

Датасет `Customer-Churn.csv` содержит 3415 записей со следующими признаками:

**Основная информация:**
- `CustomerId` — уникальный идентификатор клиента
- `Surname` — фамилия клиента
- `CreditScore` — кредитный рейтинг
- `Geography` — местоположение (страна)
- `Gender` — пол
- `Age` — возраст

**Банковские продукты:**
- `Tenure` — количество лет сотрудничества с банком
- `Balance` — баланс на счетах
- `NumOfProducts` — количество используемых продуктов
- `HasCrCard` — наличие кредитной карты (0/1)
- `IsActiveMember` — активный клиент (0/1)
- `EstimatedSalary` — прогнозируемый доход

**Дополнительные признаки:**
- `Complain` — наличие жалобы (0/1)
- `Satisfaction Score` — оценка удовлетворенности решением жалобы
- `Card Type` — тип карты
- `Point Earned` — баллы за использование карты

**Целевая переменная:**
- `Exited` — покинул ли клиент банк (0/1)
